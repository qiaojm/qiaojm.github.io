Hexo搭建（Mac）

  • 基础环境配置

    • node环境安装（建议用nvm安装）
    • npm
    • Git
    • hexo

  • 初始化一个hexo项目

    • hexo文档（https://hexo.io/zh-cn/docs/）

  • 在Github上创建xxx.github.io的版本库（xxx为用户名）

  • 关联Github

    • 配置SSH（配置了才能使用SSH地址上传本地代码到Github）

      • ssh-keygen -t rsa -C “邮箱地址”（本地创建key到/Users/iqiao/.ssh/id_rsa）
      • cd 到/Users/iqiao/.ssh文件
      • cat id_rsa.pub打开文件，并将内容拷贝到github上新建的SSH中

    • 配置本地hexo项目的_config.yml文件

      (注意：初始化生成的文件里就有deploy: ···，记得删除，不然会报错)

    • 执行 hexo -g

    • 执行 hexo -d

      • hexo -d后报错：ERROR Deployer not found: git
      • 解决方案：npm install –save hexo-deployer-git

  • 成功部署

    打开xxx.github.io看成果吧~
